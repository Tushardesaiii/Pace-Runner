# Poppins Font Files

To get the beautiful Poppins typography, download these font files from [Google Fonts](https://fonts.google.com/specimen/Poppins) and add them to this folder:

1. **Poppins-Regular.ttf** (weight: 400)
2. **Poppins-Medium.ttf** (weight: 500)
3. **Poppins-SemiBold.ttf** (weight: 600)
4. **Poppins-Bold.ttf** (weight: 700)
5. **Poppins-Black.ttf** (weight: 900)

## Quick Download

Visit https://fonts.google.com/specimen/Poppins and download the family, then extract the TTF files to this folder.

The app will work without these fonts, but will use the native system font instead.
